## Doctor User Stories
- **As a Doctor**, I want to securely log in so that I can access only doctor-specific features.
- **As a Doctor**, I want to publish my available time slots by date so that patients can book valid times.
- **As a Doctor**, I want to view my appointments by date so that I can manage my consultation schedule.
- **As a Doctor**, I want to create prescriptions after appointments so that patients receive treatment instructions.

## Patient User Stories
- **As a Patient**, I want to register and log in so that I can securely manage my account.
- **As a Patient**, I want to search doctors by name and specialty so that I can find relevant care.
- **As a Patient**, I want to view doctor availability so that I can choose suitable appointment times.
- **As a Patient**, I want to book appointments so that I can schedule visits.
- **As a Patient**, I want to view my booked appointments so that I can track upcoming care.
- **As a Patient**, I want to view prescriptions for my appointments so that I can follow medical advice.

## Admin User Stories
- **As an Admin**, I want to securely log in so that I can manage the system.
- **As an Admin**, I want to add/update/delete doctor accounts so that doctor records remain accurate.
- **As an Admin**, I want to run daily appointment reports by doctor so that I can monitor clinic utilization.
- **As an Admin**, I want to identify doctors with most patients by month/year so that I can support planning decisions.
