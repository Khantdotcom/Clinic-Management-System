## Doctor User Stories
- **As a Doctor**, I want to log in securely so that I can access only my professional portal.
- **As a Doctor**, I want to define my available time slots by date so that patients can book only valid appointment windows.
- **As a Doctor**, I want to view all appointments scheduled for a selected day so that I can plan consultations efficiently.
- **As a Doctor**, I want to create prescriptions for completed appointments so that patients can follow treatment guidance.

## Patient User Stories
- **As a Patient**, I want to register and log in so that I can manage my medical appointments securely.
- **As a Patient**, I want to search doctors by name and specialty so that I can find the right physician quickly.
- **As a Patient**, I want to check doctor availability before booking so that I can choose a convenient time.
- **As a Patient**, I want to book an appointment and receive confirmation so that I know my visit is scheduled.
- **As a Patient**, I want to view my upcoming and past appointments so that I can track my care history.
- **As a Patient**, I want to view prescriptions tied to my appointments so that I can follow prescribed medications.

## Admin User Stories
- **As an Admin**, I want to log in with elevated permissions so that I can administer system settings and users.
- **As an Admin**, I want to add, update, and remove doctor accounts so that provider records stay accurate.
- **As an Admin**, I want to run daily appointment reports by doctor so that I can monitor operational activity.
- **As an Admin**, I want to identify doctors with most patients by month/year so that I can support capacity planning.
